# 🎓 College Management System (MySQL)

## 📌 Project Overview

The College Management System is a relational database project developed using MySQL. It is designed to manage students, teachers, departments, courses, enrollments, results, and fee records.

This project demonstrates SQL database design, data manipulation, and data analysis techniques commonly used in real-world database systems.

---

## 🚀 Features

- Relational Database Design
- Primary & Foreign Keys
- Student Management
- Teacher Management
- Department Management
- Course Management
- Enrollment System
- Result Management
- Fee Management
- SQL Data Analysis
- Aggregate Functions
- JOIN Queries
- GROUP BY & HAVING
- Views
- Stored Procedures
- Business Reports
